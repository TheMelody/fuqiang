2、How do we use permissions

（1）、Permission: android.permission.SYSTEM_ALERT_WINDOW used before android 6.0, floating window permission, used for screen edge detection full screen gestures, to help users achieve edge sliding back to the upper page.

（2）、Permission: android.permission.SYSTEM_OVERLAY_WINDOW after android 6.0 is used, the floating window permission is used to detect the full screen gestures on the edge of the screen, and help users to realize the edge sliding back to the upper page.

（3）、Permission: android.permission.RECEIVE_BOOT_COMPLETED is used to monitor the system startup broadcast, and help to detect whether the gesture still exists after the boot.

（4）、Permission: android.permission.BIND_ACCESSIBILITY_SERVICE accessibility service, used to realize the function of sliding back on the edge of the full screen, this function will not monitor the screen content.

（5）、Permission: android.permission.CAMERA, Used to realize, control center "mirror" function, and devices compatible with android 6.0 are used to turn on the flash.

（6）、Permission: android.permission.RECORD_AUDIO,Used to realize the "screen recording" function of the control center, used to record microphone audio
